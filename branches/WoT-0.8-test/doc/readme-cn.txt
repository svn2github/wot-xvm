1. 安装 Microsoft .NET Framework 2.0.

2. 安装 Dokan 运行库: http://dokan-dev.net/en/download/#dokan

3. 解压缩压缩包到WOT根目录

4. 设置插件配置文件 \res_mods\<game version>\gui\flash\XVM.xvmconf
   可以参考\xvm-doc\samples\目录下的示例文件，
   或者访问http://bulychev.net/generator/在线生成此文件。
   注意：如果你需要手动修改配置文件，不要使用MS Word。可以使用“记事本”、“写字板”等文本编辑器编辑。

5. 使用wot-xvm-proxy.exe启动游戏.

   如果你需要启动WOT Launcher，请执行如下命令：   
     wot-xvm-proxy.exe /launcher


注意：
     如果你在使用Skype，请修改你的端口设置：“工具”->“选项”->“高级”->“连接”，确认“使用80和443端口...”没有被选中。
