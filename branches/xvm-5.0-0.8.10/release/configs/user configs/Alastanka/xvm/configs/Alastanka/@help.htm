18.06.2013 		XVM: eXtended Visualisation Mod Официальный сайт. Скачивание и документация Исходники XVM: Последние изменения в файлах нового конфига
XVM: Редактор NEW конфигураций @xvm.xc специально для xvm.alastanka.org.ua
Для правильной работы разместить этот файл в папку с файлом @xvm.xc пример мой: \World_of_Tanks\res_mods\xvm\configs\user configs\Alastanka\@xvm.xc, кликнув на нужный файл он откроется для редактирования.
1@xvm.xc Главный конфигурационный файл.
  • Версия конфига
  • Версия редактора
  • Общие параметры конфига. Все параметры информационные, в моде не используются
    • Автор конфига
    • Описание конфига
    • Адрес, где выкладываются обновления конфига
    • Дата последней модификации конфига
    • Поддерживаемая версия игры
    • Минимально необходимая версия мода XVM
  • Ссылки на все файлы конфига(Все настройки вынесены в отдельные файлы)
  • * Коффициенты затемнения значков класса техники "consts", вынесен Alastanka в (consts.xc)
2alpha.xc Настройки динамической прозрачности. Значения от меньшего к большему.
  • Динамическая прозрачность по оставшемуся здоровью
  • Динамическая прозрачность по проценту оставшегося здоровья
  • Динамическая прозрачность по эффективности
  • Динамическая прозрачность по проценту побед
  • Динамическая прозрачность по количеству кило-боев
3battle.xc Общие параметры боевого интерфейса.
  • отключить зеркалирования иконок танков
  • отключить всплывающую внизу панель после смерти.
  • убрать переключатель режимов ушей мышкой.
  • отключить подсветку иконки своего танка и взвода.
  • Формат часов на экране панели отладки (возле FPS)
  • использовать стандартные маркеры клиента
  • Путь к папке иконок кланов относительно
  • Панель счёта в бою
  • Внутриигровая панель критов от навыка "экспет"
  • * Двигать визуальные элименты "elements", вынесен Alastanka в (battleElements.xc)
3abattleElements.xc * * Двигать визуальные элименты.
  • * Дополнительный файл сборки Alastanka. Подгружается с файла (battle.xc)
4battleLoading.xc Параметры экрана загрузки боя.
  • Формат часов на экране загрузки боя.
  • включить отображение шансов на победу
  • Показывать экспериментальную формулу расчета шансов
  • убрать отображение иконки взвода. На пустое поле можно вывести, например, иконку клана
  • Параметры отображения иконки игрока/клана
  • отображать иконки игрока/клана в окне загрузки боя
  • Формат отображения для левой панели
  • Формат отображения для правой панели
5captureBar.xc Полоса захвата.
6colors.xc Настройки цветов.
  • Системные цвета.
  • Динамический цвет по типу урона
  • Динамический цвет по типу техники
  • Динамический цвет по оставшемуся здоровью
  • Динамический цвет по проценту оставшегося здоровья
  • Динамический цвет по шкале XVM
  • Динамический цвет по эффективности
  • Динамический цвет по рейтингу WN6
  • Динамический цвет по рейтингу TEFF (E)
  • Динамический цвет по проценту побед
  • Динамический цвет по TWR (T-Calc)
  • Динамический цвет по количеству кило-боев
  • Динамический цвет по количеству боев на текущем танке
  • Динамический цвет по среднему урону за бой на текущем танке
  • Динамический цвет по эффективности урона за бой на текущем танке
  • Динамический цвет по среднему количеству фрагов за бой на текущем танке
  • Динамический цвет по среднему количеству засвеченных врагов за бой на текущем танке
  • * Настройки цвета для урона(у Alastanka вынесен в отдельный файл colorsDamage.xc)
6аcolorsDamage.xc * * Настройки цвета для урона.
  • * Дополнительный файл сборки Alastanka. Подгружается с файла "colors.xc"
6бconsts.xc * * Коффициенты затемнения значков класса техники.
  • * Дополнительный файл сборки Alastanka. Подгружается с файла (@xvm.xc)
7finalStatistic.xc Параметры окна послебоевой статистики.
8hangar.xc Параметры ангара.
  • скрыть иконку обучения
  • Пинг серверов
  • Окно достижений
    • включить фильтр отображения танков в ангаре по умолчанию
    • показывать колонку эффективности по танку
9hitLog.xc Лог попаданий (счетчик своих попаданий).
10hotkeys.xc Специальные горячие клавиши XVM.
11iconset.xc Наборы иконок. Пути относительно res_mods/xvm/res.
12login.xc Параметры экрана логина.
  • Пинг серверов
13markers.xc Маркеры над танками. (Все настройки вынесены в отдельные файлы).
14markersAliveExtended.xc Настройки маркеров для живых с Alt.
15markersAliveNormal.xc Настройки маркеров для живых без Alt.
16markersDeadExtended.xc Настройки маркеров для трупов с Alt.
17markersDeadNormal.xc Настройки маркеров для трупов без Alt.
18minimap.xc Общие параметры миникарты.
  • Увеличение миникарты по нажатию кнопки
  • Квадрат со стороной 1000m
19minimapCircles.xc Круги на миникарте. Дистанция только в реальных метрах карты. Только для своей техники.
20minimapLabels.xc Надписи на миникарте. Поддерживают HTML.
21minimapLines.xc Линии на миникарте. Только для своей техники.
22playersPanel.xc Параметры панелей игроков ("ушей").
  • Маркер статуса засвета в правой боковой панели списка игроков
  • Режим ушей "medium" - первые средние уши в игре
  • Режим ушей "medium2" - вторые средние уши в игре
  • Режим ушей "large" - широкие уши в игре
23rating.xc Блок управлением статистикой (только с xvm-stat).
  • включить отображение статистики. Глобальный переключатель - без него модуль статистики будет везде отключен
  • включить загрузку данных по противникам в "тумане войны"
  • включить сохранение статистики в файл "xvm-stat.log"
  • включить отображение статистики в достижениях пользователя
  • включить отображение статистики при наборе роты
24squad.xc Параметры окна взвода.
25statisticForm.xc Параметры окна статистики по клавише Tab.
26texts.xc Текстовые подстановки.
  • Текст для макроса {{vtype}}
27turretMarkers.xc Отображаемые строки {{turret}} маркера.
  • Сток/Топ Башня/Орудие
28vehicleNames.xc Замена названий танков.
