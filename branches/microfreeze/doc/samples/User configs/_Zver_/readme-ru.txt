Конфиг _Zver_ (называется так, но поддерживается только лично мной).
На данный момент полностью совместим с XVM Full 3.1.2 test 4.



- 2 версии: на основе процента побед и эффективности.
- Маркер танка: узкая полоска жизни, слева хп, справа %, снизу основная статистикиа (в зависимости от версии), сверху название танка.
- Дополнительно по альту: иконка и уровень танка, вся остальная статистика (в зависимости от версии), ник игрока.
- Ширина широких ушей 100, альфа 50, альфа иконок танков и взводов 70.
- Включены часы и шансы на победу, взводы в широких ушах включены, кнопки слева вверху отключены.
- Клановые/пользовательские иконки 24х24 на маркерах танков и в широких ушах за дефаултными иконками танков.
- В загрузочном экране общая статистика, по табу - по танку. Из-за того, что всё не влезает в экран.

Различия в версиях:
% побед:
Уши: % побед.
Основной маркер: % побед по танку, кол-во боёв по танку.
Расширенный маркер: общий % побед, эффективность, общее кол-во боёв.
Эффективность:
Уши: эффективность.
Основной маркер: эффективность.
Расширенный маркер: общий % побед, % побед по танку, кол-во боев по танку.
Примечание: Если кол-во боёв по танку меньше 100, % побед по танку выводится как среднее между % побед по танку и общим % побед.

Versions difference:
Winratio:
Sides: winratio.
Marker: current vehicle winratio, current vehicle battles.
Alternative marker: average winratio, efficiency, total battles.
Efficiency:
Sides: efficiency.
Marker: efficiency.
Alternative marker: overall winratio, current vehicle winratio, current vehicle battles.
Note: If current vehicle battles are less than 100, you can see average of current vehicle battles and total battles.

В случае крякозябров/квадратов попробуйте установить шрифт из архива Font_consolas.zip
In case any problems with fonts you have to install 'consolas' font into your system. You can find it in Font_consolas.zip
