1. ОБЩАЯ ИНФОРМАЦИЯ
-----------------------------------------------------------

  Эта программа используется для отображения статистики в моде XVM

  Сайт проекта: http://code.google.com/p/wot-xvm/
  Поддержка:    http://www.koreanrandom.com/forum/forum/43-xvm
  FAQ:          http://www.koreanrandom.com/forum/topic/1381-faq

-----------------------------------------------------------
2. УСТАНОВКА
-----------------------------------------------------------

  1. Необходимо установить XVM (http://code.google.com/p/wot-xvm/).

  2. Игра должна быть установлена на диск с файловой системой NTFS.

  3. Должен быть установлен Microsoft .NET Framework 3.5 Client Profile.
     Можно проверить установленные версии .NET с помощью этой утилиты:
       http://www.tmgdevelopment.co.uk/versioncheck.htm

  4. Установить DokanLibrary: http://dokan-dev.net/en/download/#dokan

  5. Разархивировать архив в папку с игрой:
     Правой кнопкой на архив -> "Извлечь все..." -> выбрать папку игры -> "Извлечь"

  6. Для отображения статистики необходимо запускать файл xvm-stat.exe (игра
     будет запущена автоматически).

     Если необходимо отображать статистику и запускать игру через лаунчер,
     запустите xvm-stat.exe с аргументом командной строки /launcher:
       Создайте ярлык на файл xvm-stat.exe
       Откройте Свойства ярлыка
       В поле Объект напишите 'xvm-stat.exe /launcher' (без кавычек)
       Нажмите ОК

  7. Если стоит Скайп, необходимо в
       инструменты -> надстройки -> дополнительно -> соединение -> УБРАТЬ ГАЛОЧКУ
       "Использовать порты 80 и 443 в качестве входящих альтернативных".

  8. Если необходимо отправить отчет об ошибке, приложите к отчету лог-файл
     XVM.log из папки игры.
