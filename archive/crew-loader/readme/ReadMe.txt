Описание : 

Мод выполняет автоматическую загрузку экипажа в танк.

Добавлено 2 варианта : 
							*	Посадить родной экипаж
							*	Посадить лучший из доступных экипажей

В ангаре, выберите танк в который вы хотите посадить экипаж.

Правой кнопкой мышки щелкаем по пустому слоту танкиста и в появившемся меню нажимаем на интересующую кнопку.

 -	Верхняя кнопка 	: Поместить родной экипаж - производится посадка экипажа в танк, только в пустые слоты. 
												Если родной экипаж не найден то посадки не совершается.
 -	Нижняя кнопка 	: Поместить лучший экипаж - происходит выбор лучшего доступного экипажа для пустых слотов данного танка.

						Приоритет отдаётся танкисту с наибольшим основным опытом при посадке в данный танк. 
						При равенстве данного опыта у двух кандидатов, проводится сравнение количества перков и уровня последнего перка при равном количестве.

На каждое место в экипаже рассматривается список доступных кандидатов.

Установка : 

Мод нужно устанавливать в папку по адресу res_mods\0.8.3\gui\flash\
По дефолту это C:\Games\World_of_Tanks\res_mods\0.8.3\gui\flash\

Чтобы добавить названия кнопок в меню, нужно положить файл в папку  с текстом для ГУИ. Перед установкой не забываем сделать копию оригинального файла.
\res\text\LC_MESSAGES\menu.mo

2 варианта названий кнопок на выбор :
- файл menu.mo 		: Кнопки названы - Родной / Годный Экипаж 
- файл menu_rm.mo 	: Кнопки названы - Suigintou / Suiseiseki (перед установкой переименовать в menu.mo)

Альтернативно одарённый вариант установки : кидаем всю папку \res\text\ по адресу \res_mods\0.8.3\text и затем уже в скопированную папку кидаем файл menu.mo
