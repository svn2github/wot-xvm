Описание изменений.

Экран загрузки боя
  Тут ничего не менялось, никакой лишней информации, все как в стандарте.

Панели игроков
  Так как я играю на широкоформатнике, то использую панели только в широком режиме.
  Ширина панелей выставлена так, чтобы помещались ники средней длины.
  Поле с иконками взводов отключено.
  По краям панелей слева и справа выведена боевая эффективность игроков, которая раскрашена
  в три цвета в зависимости от ее значений(<900-красный; <1200-желтый; <1200-зеленый).

Окно статистики
  Боевая эффективность расскрашенная в зависимости от ее значения
  Количество боев у игроков в килобоях с динамическим изменением цвета (<4к - красный; <8к-желтый; <100к-зеленый)
  Общий процент побед с динамическим изменением цвета (<48% - красный; <53%-желтый; <101%-зеленый)

Маркеры
  Настройки для своих и противников идентичны, поэтому скриншоты буду приводить для одного варианта.

Нормальный режим
  Живой - Отображается название танка, узкая полоска ХП, количество ХП.
  Мертвый - Отображается только маркер типа танка

Расширенный режим
  Живой - Отображается ник игрока, узкая полоска ХП, количество ХП.
  Мертвый - Отображается ник игрока, название танка.

В конфиге используется шрифт Cuprum (если вместо текста квадраты, нужно установить)
