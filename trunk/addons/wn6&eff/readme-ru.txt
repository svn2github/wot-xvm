Автоматические скрипты для замены одного представления рейтинга на другое в конфигурационных файлах XVM.

Использование:

1 вариант:
файл скрипта может лежать в произвольном месте.
Выделяем файл конфига (или несколько файлов конфига) и перетаскиваем его на файл скрипта.
Всё, внутри файла (файлов) конфига рейтинг изменен.

2 вариант:
помещаем скрипт в одну папку с конфиг файлом XVM.xvmconf, запускаем скрипт.
Всё, внутри файла конфига рейтинг изменен.
