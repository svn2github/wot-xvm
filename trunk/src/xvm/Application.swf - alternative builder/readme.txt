This folder is for Windows users.

Install latest RABCDAsm from http://files.thecybershadow.net/RABCDAsm/ , add to PATH evironment path to its binaries.
Install patch.exe from http://sourceforge.net/projects/gnuwin32/files/patch/2.5.9-7/patch-2.5.9-7-setup.exe/download .
Add to PATH environment c:\Program Files (x86)\GnuWin32\bin (for 64bit systems) or c:\Program Files\GnuWin32\bin (for 2bit systems).
Use included bat file to build patced SWF.

No more need to cygwin or other unix-style big sets of tools.
