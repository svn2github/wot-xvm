    отключение зеркалирования иконок и прозрачность)
  * версия конфига изменена на 1.2.0. Перемещены параметры:
      rating/battleLoading => battleLoading
      rating/statisticForm => statisticForm
      rating/playersPanel => playersPanel
      battle/battleLoadingShowClock => battleLoading/showClock
      battle/playersPanelAlpha => playersPanel/alpha
      battle/playersPanelLargeWidth => playersPanel/large/width
  * настраиваемые системные цвета (секция colors/system в XVM.xvmconf)
    работает для всех элементов, кроме маркеров типа танка
  * настраиваемые форматы полей имени игрока и названия танка в ушах
  * темный цвет для мертвых или не загруженных игроков в ушах
  * настройка прозрачности иконок в ушах (параметр playersPanel/iconAlpha)
  * изменение размера полей в "ушах"
  * автоматический выбор самого быстрого сервера (в данный момент работает
    только для китайских серверов)
  * добавлены наборы иконок (секция конфига "iconset")
  * добавлены иконки игрока/клана
  * добавлен расчет шансов на победу.
    Обсуждение: http://forum.bulychev.net/viewtopic.php?f=4&t=26
