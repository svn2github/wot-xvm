Последнее обновление 24.10.12

Версия xvm-full-3.0.4.pre2 (WoT client 0.8.0.)

Конфиг расчитан на альтернативные иконки (любые, в которых присутствует название танка)

- Из экрана загрузки и ТАВ убрано название танка, для выравнивания столбцов статистики применен шрифт Droid Sans Mono (присутствует в архиве);
- Для отображения символов конфига создан специальный шрифт XVM Symbol. Шрифт содержит используемые символы из других шрифтов, а так же символы, созданные специально для конфига. По мере необходимости шрифт будет дополняться;
- По ТАВ, в "ушах" и в экране загрузки- общая статистика, в маркере по Alt- общая и по танку;
- В широких "ушах" вместо процента побед показывается общее количество боев, вместо названия танка- коэф. эффективности, в средних ушах ники раскрашены по эффективности, во вторых средних ушах вместо названия танка показывается коэф. эффективности;
- Переработана цветовая гамма статистики;
- Скорректированы системные цвета живых союзников, противников взводных;
- Уменьшена ширина поля имени игрока в широких "ушах", увеличена прозрачность "ушей";
- В маркерах узкая зелёная/красная полоска жизни с полупрозрачной подложкой, снизу текущее здоровье, сверху название танка;
- В маркерах без Alt добавлены символы  у союзников и  у противников, окрашенные в цвет эффективности игрока;
- По Alt относительно полосы здоровья:
  слева- % побед на танке,
  справа- общий % побед,
  сверху- количество гектобоев на танке (гекто (h) = 100), эффективность, общее количество килобоев, уровень и название танка, ник,
  снизу- %здоровья / максимальное здоровье. Каждый элемент статистики раскрашен в собственный цвет;
- Маркер стоковой башни: если танк имеет стоковую башню, на которую можно поставить топ орудие, отображается символ  , если у танка стоковая башня, на которую топ орудие поставить невозможно, отображается символ  ;
- Видоизменен хит-лог;
- Изменено цветовое оформление отлетающего урона. При обычном уроне красится тень в зависимости от сочетания нанесшего / получившего урон. У последнего урона аналогично красится текст. Свой урон- белый с черной тенью;
- Изменено оформление смертельного урона;
- У мертвых показывается только символ класса танка, по альт добавляется его название и ник игрока;
- Отключена зеркальность иконок;
- Клановые иконки везде выключены;
- Шансы на победу включены;
- Всплывающая внизу панель после смерти выключена;
- Подсветка иконки своего танка и взвода (уши, ТАВ, загрузка) выключена.
