Сделано на базе конфига от rreeedd к версии 1.7, за что ему отдельное спасибо.
Особенности:
-На игровом экране только процент побед, раскраска ников тоже по проценту побед, несколько упрощённая по сравнению с дефаултной.
-По табу можно дополнительно посмотреть кол-во боев.
-Коэффициент эффективности отсутствует полностью.
-Ширина широких ушей 100, альфа 50.
- На танках узкая зелёная/красная полоска жизни, слева хп, справа %, снизу %побед, сверху название танка, по альту - ник игрока.
