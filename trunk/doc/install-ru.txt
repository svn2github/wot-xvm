1. Игра должна быть установлена на диск с файловой системой NTFS.
2. Должен быть установлен Microsoft .NET Framework 2.0.
   Можно проверить установленные версии .NET с помощью этой утилиты:
     http://www.tmgdevelopment.co.uk/versioncheck.htm
3. Установить DokanLibrary: http://dokan-dev.net/en/download/#dokan
4. Разархивировать архив в папку с игрой:
   Правой кнопкой на архив -> "Извлечь все..." -> выбрать папку игры -> "Извлечь"
5. Если необходимо, настроить конфигурационный файл:
     \res_mods\<game version>\gui\flash\XVM.xvmconf
   Можно выбрать готовый из папки \xvm-doc\samples\, или воспользоваться
   онлайн-редактором: http://bulychev.net/generator/
   Старые конфигурационные файлы OTMData.xml тоже поддерживаются, вы можете
   их использовать если вам не нужны новые возможности.
   Все возможные настройки можно посмотреть в этом файле:
     \xvm-doc\samples\Full config\XVM.xvmconf
   Внимание: По умолчанию статистика выключена, поэтому если она нужна, включите
   ее в редакторе, или возьмите готовый конфигурационный файл.
   Внимание: Если вы меняете конфиг вручную, используйте Блокнот (notepad),
   НЕ используйте word, wordpad и подобные редакторы)
6. Если статистика не нужна, запускайте игру как обычно.
   Для отображения статистики необходимо запускать файл wot-xvm-proxy.exe (игра
   будет запущена автоматически).
   Если необходимо отображать статистику и запускать игру через лаунчер,
   запустите wot-xvm-proxy.exe с аргументом командной строки /launcher:
     Создайте ярлык на файл wot-xvm-proxy.exe
     Откройте Свойства ярлыка
     В поле Объект напишите 'wot-xvm-proxy.exe /launcher' (без кавычек)
     Нажмите ОК
7. Если стоит Скайп, необходимо в
     инструменты -> надстройки -> дополнительно -> соединение -> УБРАТЬ ГАЛОЧКУ
     "Использовать порты 80 и 443 в качестве входящих альтернативных".
8. Если необходимо отправить отчет об ошибке, запустите wot-xvm-proxy.exe с
   ключом /debug и добавьте вывод на консоль в отчет.
