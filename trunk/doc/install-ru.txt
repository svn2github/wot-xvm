1. Должен быть установлен Microsoft .NET Framework 2.0.
2. Установить DokanLibrary: http://dokan-dev.net/en/download/#dokan
3. Разархивировать архив в папку с игрой
4. Настроить конфиг \res_mods\<версия игры>\gui\flash\XVM.xvmconf
   Можно выбрать готовый из папки \xvm-doc\samples\, или воспользоваться
   генератором: http://bulychev.net/generator/
   Внимание: Если вы меняете конфиг вручную, используйте Блокнот (notepad),
   НЕ используйте word, wordpad и подобные редакторы)
5. Для игры запускать файл wot-xvm-proxy.exe.
   Если необходимо запускать игру через лаунчер, необходимо запускать с ключом /launcher:
     wot-xvm-proxy.exe /launcher
   Можно сделать необходимые ярлыки на рабочем столе для упрощения запуска.

Если стоит Скайп, необходимо в инструменты - надстройки - дополнительно - соединение
УБРАТЬ ГАЛОЧКУ "Использовать порты 80 и 443 в качестве входящих альтернативных".
