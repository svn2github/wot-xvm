UPD: [30.10.2012] Добавлена новая версия конфига.

Изменения:
Экран загрузки боя
Иконки игрока/клана вынесены за иконки взвода. Размер 16х16.
Статистика выравнина на 3-4 символа.

Живые без Alt
Показывается: полоска хп, процент и текущее хп, название танка, иконка типа танка.
"Уши" стандартные без изменений, иконка игрока/клана вынесена за границу иконки танков. Размер 16х16.

Живые с Alt
Показывается: ник игрока, иконка танка, иконка типа танка, статистика эффективности игрока.
"Уши" стандартные без изменений, иконка игрока/клана вынесена за границу иконки танков. Размер 16х16.
Добавлены иконки игрока/клана в маркерах. Размер 16х16.
Процент побед и гекто-бои на конкретном танке.

Окно статистики
Иконки игрока/клана вынесены за иконки взвода. Размер 16х16.
Статистика выравнина на 3-4 символа.

Урон по союзнику и пративнику
Урон красного цвета по свои.
Урон синего цвета по врагу.

Смерть союзника
Формат: ник союзника.

Смерть противника
Формат: ник и танк противника.

Мёртвый без Alt
Показывается только иконка типа танка.

Мёртвый с Alt
Показывается иконка типа танка и название танка, для ориентации на поле боя.
Добавлены иконки игрока/клана в маркерах. Размер 16х16.
